# ai-cli info

### bash script for AI daily tasks

1) CLI wrapper for Claude Code y Copilot CLI 

2) Context orchestrator 

3) Shortcuts for common tasks: debug, feature, refactor, review, planning. 

4) Contacts: source + context + model + prompt

## Steps

1) Install Claude Code CLI and GitHub Copilot CLI
2) Save ai-cli.sh file into your /Users/<Your User>/bin folder
3) Create .ai-private with these files () in your root project:

/CLAUDE.md"
.ai-private/FEATURE.md"
.ai-private/DEBUG.md"
.ai-private/REFACTOR.md"
.ai-private/REVIEW.md"
.ai-private/PLANNING.md"

4) run in your terminal: ai-cli
